Security & Safety

DTHBoost is open source. Every line of code is public. Here is exactly what it does — and does not do.

✓ What DTHBoost Does

  • ✓ Change Windows power plans via powercfg (Microsoft documented API)
  • ✓ Toggle registry values under HKCU\Software\Microsoft\GameBar and GameDVR
  • ✓ Set GPU preferences via HKCU\Software\Microsoft\DirectX\UserGpuPreferences
  • ✓ Run Intel PresentMon to capture frame timing data (read-only)
  • ✓ Detect running processes via tasklist (standard Windows tool)
  • ✓ Detect hardware via WMI queries (Win32_Processor, Win32_VideoController)
  • ✓ Create a rollback snapshot before every change
  • ✓ Auto-restore all settings when you close the game

✗ What DTHBoost NEVER Does

  • ✗ DLL injection into any process — blocked by design
  • ✗ Read or write game memory — would trigger anti-cheat
  • ✗ Modify or suspend anti-cheat services (Vanguard, Easy Anti-Cheat, BattlEye)
  • ✗ Install kernel drivers — not needed, not included
  • ✗ Macros, recoil control, or input automation
  • ✗ Disable Windows Defender entirely (only adds game folder exclusion)
  • ✗ Modify BCDEdit, HPET, or boot configuration
  • ✗ Network traffic interception or packet manipulation
